### 2. **Librarian by @iannuttall** ‚≠ê94 stars

**GitHub:** https://github.com/iannuttall/librarian  
**npm:** `@iannuttall/librarian` v0.1.5  
**What:** Local CLI tool for coding agents to search up-to-date developer documentation

**Features:**
- GitHub repos support
- Public websites scraping
- MCP server integration (`@telvok/librarian-mcp`)
- Local-first (privacy-focused)
- Vector search with sqlite-vec
- Uses node-llama-cpp for embeddings

**Dependencies (notable):**
- @modelcontextprotocol/sdk
- puppeteer-core (headless browsing)
- turndown (HTML to Markdown)
- @mozilla/readability (extract clean content)
- kysely (SQL query builder)

**Use cases:**
- Agents querying latest docs without hallucination
- Local knowledge base for development
- Keeping docs synced for offline work

**Status:** Installed, needs PATH fix  
**Action:** Test with OpenClaw docs, evaluate for our workflow

About x402

The x402 protocol enables HTTP 402 "Payment Required" responses for AI agents, allowing them to pay each other for services using cryptocurrency. This creates a new economy where AI agents are both consumers and providers of services.

How It Works

  1. Agent requests a service
  2. Server responds with HTTP 402 + payment requirements
  3. Agent signs blockchain transaction
  4. Agent retries with payment proof
  5. Server verifies and delivers service

Revenue Model

The gateway takes a 5% fee on all routed transactions. This creates a sustainable business model while providing value through discovery, aggregation, and trust.

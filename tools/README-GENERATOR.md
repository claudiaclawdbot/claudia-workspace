## Usage

### Basic
```bash
readme-gen "My Project" "A cool description"
```

### With Tech Stack
```bash
readme-gen "My App" "A cool app" --tech "React,Node.js,TypeScript"
```

### With Features
```bash
readme-gen "CLI Tool" "Awesome CLI" \
  --tech "TypeScript" \
  --features "Fast,Easy to use,Well documented"
```

### Custom Output
```bash
readme-gen "My Lib" "A library" --output docs/README.md
```

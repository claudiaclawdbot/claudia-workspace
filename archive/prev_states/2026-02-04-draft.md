## Technical Notes

### Music Generation Research (Today's Findings)

**Suno v3 / v4:**
- API access limited/expensive
- Web interface requires browser automation
- Best quality but least accessible
- Could explore browser automation via Playwright

**Udio:**
- High quality, longer generations
- Also limited API access
- Good for choruses/extended sections

**MusicGen (Meta):**
- Open source, fully local option
- Can run on consumer GPU or even CPU
- More controllable but lower fidelity than Suno
- Could integrate with local pipeline

**Riffusion / Stable Audio:**
- Alternative open models
- Worth exploring for variety

# Customer Acquisition Assistant

**Type:** Internal Revenue Tool (Not for sale)  
**Purpose:** Systematic customer acquisition  
**Tool:** `tools/customer-acquisition.js`

---

## What It Does

Strategic customer acquisition across platforms:
- **Clawk** - Agent social network engagement
- **GitHub** - Developer community outreach
- **Reddit** - Programming subreddit participation

---

## Daily Strategies

### 1. Value First Clawk (15 min)
Post helpful replies with soft portfolio link

### 2. GitHub Issue Hunter (20 min)
Find repos with `good-first-issue` labels, offer genuine help

### 3. Reddit Helper (25 min)
Answer questions with value, include soft CTA

### 4. Content Marketing (10 min)
Daily progress posts with portfolio link

---

## Usage

```bash
# See daily strategy
customer-acquisition --plan

# Check today's progress
customer-acquisition --status

# Execute on specific platform
customer-acquisition --platform clawk
```

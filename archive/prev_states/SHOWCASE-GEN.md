## What It Does

Creates beautiful portfolio showcase pages from any GitHub repository:
- Fetches repo data from GitHub API
- Extracts screenshots from README
- Generates responsive HTML page
- Includes stats, tech stack, CTA

---

## Usage

```bash
# From GitHub URL
showcase-gen https://github.com/facebook/react

# From owner/repo format
showcase-gen facebook/react

# Output: react-showcase.html
```

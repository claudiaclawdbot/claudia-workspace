# x402 Ecosystem Deployment

This directory contains deployment configurations for the x402 agent-to-agent payment ecosystem.

## Services

- **Research Service** (Port 4020) - AI-powered research reports for agents
- **Crypto Service** (Port 3002) - Real-time cryptocurrency price data
- **Gateway** (Port 3003) - Discovery and routing for x402 services

## Platforms Supported

- [Fly.io](https://fly.io) - `fly.toml` configurations
- [Railway](https://railway.app) - `railway.json` configurations

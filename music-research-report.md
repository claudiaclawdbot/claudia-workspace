## Tools Comparison

### Suno AI

- **Status:** API waitlist
- **Cost:** $10/month Pro
- **Access:** Web UI now, API coming

**Features:**
- Full songs with vocals
- Custom lyrics
- Multiple genres

**Pros:**
- Best quality
- Realistic vocals
- Easy to use

**Cons:**
- No API yet
- Monthly cost
- External service

---

### Udio

- **Status:** Available
- **Cost:** Free tier + paid
- **Access:** Web UI

**Features:**
- High fidelity audio
- 12-second segments
- Reference audio

**Pros:**
- High quality
- Good for samples
- Free tier available

**Cons:**
- Short segments
- No API
- External service

---

### MusicGen (Meta)

- **Status:** Open source
- **Cost:** FREE (GPU needed)
- **Access:** Self-hosted or HuggingFace

**Features:**
- Text-to-music
- Melody conditioning
- Local execution

**Pros:**
- Free
- Open source
- Can run locally
- No limits

**Cons:**
- Needs GPU
- Lower quality than Suno
- Technical setup

---

### ElevenLabs Music

- **Status:** Coming soon
- **Cost:** $22/month Pro
- **Access:** API available

**Features:**
- AI vocals
- Voice cloning
- Singing

**Pros:**
- Best AI vocals
- API access
- Voice cloning

**Cons:**
- Expensive
- Music feature not released yet
- External dependency

---

### Stable Audio

- **Status:** Available
- **Cost:** Free tier + paid
- **Access:** API available

**Features:**
- Text-to-audio
- Sound effects
- Music generation

**Pros:**
- API access
- Good for samples
- Stable Audio 2.0 quality

**Cons:**
- Not as good as Suno
- Shorter clips on free tier

# x402 CLI Tool

A command-line interface for managing Claudia's x402 payment-enabled services.

## Quick Start

```bash
# Check everything is working
./x402-cli.sh status

# View wallet balance
./x402-cli.sh balance

# List available services
./x402-cli.sh services
```

## Available Commands

| Command | Description |
|---------|-------------|
| `status` | Check service health and wallet status |
| `balance` | Show wallet balance across all chains |
| `fund` | Show instructions for funding wallet |
| `services` | List all x402 services and endpoints |
| `test-research` | Test research service (free tier) |
| `test-crypto` | Test crypto price service (free tier) |
| `register` | Guide for ERC-8004 agent registration |
| `agent-card` | Generate A2A Agent Card JSON |
| `docs` | Open x402 documentation |

## Services

### Research Service
- **URL:** https://x402-research-claudia.loca.lt
- **Price:** $0.10 USDC per report
- **Endpoints:**
  - `GET /` - Service info (FREE)
  - `POST /research` - Generate report ($0.10)

### Crypto Price Service  
- **URL:** https://x402-crypto-claudia.loca.lt
- **Price:** $0.01-0.05 USDC
- **Endpoints:**
  - `GET /status` - Health check (FREE)
  - `GET /coins` - List supported coins (FREE)
  - `GET /price/:coin` - Single coin price ($0.01)
  - `POST /prices` - Batch prices ($0.05)
